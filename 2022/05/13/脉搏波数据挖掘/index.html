特征提取

根据脉搏波的成因和高血压的病理特征，高血压与脉搏波的时域特征有很密切的关系，因此本次特征挖掘选择了十四种时域特征、

  • mean_data # 均值
  • min_data # 最小值
  • max_data # 最大值
  • PTP_data # 峰峰值
  • FGFZ_data # 方根幅值
  • var_data # 方差
  • std_data # 标准差
  • RMS_data # 均方根
  • skew_data # 偏度
  • kur_data # 峭度
  • cresF_data # 峰值因子
  • clearF_data # 裕度因子
  • shapeF_data # 波形因子
  • impF_data # 脉冲指数

通过对exls文件中的周期按行读入，然后以每行为单位分别求时域特征，最后加入该时域特征的数组中。

归一化处理

归一化处理是保留所有的特征，但是减少参数的大小（或者是说：减少参数的重要性）

因此为了消除指标之间量纲的影响，需要进行数据标准化处理，以解决数据指标之间的可比性。原始数据经过数据标准化处理后，各指标处于同一数量级，适合进行综合对比评价。

使用热力图分析特征相关性

变量的关系可分为两种：

两个及以上变量间相互影响——平行关系
一个变量变化受另一个变量的影响——依存关系

它们对应的分析方法：

相关分析是研究呈平行关系的相关变量之间的关系
回归分析是研究呈依存关系的相关变量之间的关系

简单相关分析即就是通过对大量数字资料的观察，消除偶然因素的影响，探求现象之间相关关系的密切程度和表现形式

所有相关分析中最简单的就是两个变量间的线性相关，一变量数值发生变动，另一变量数值会随之发生大致均等的变动，各点的分布在平面图上大概表现为一直线。

在这里对特征之间的关系做简单的判断，使用zeros_like做判断，使用seaborn生成热力图如图所示

由图可以看出，血压与峭度和偏度的关联性不大，和其他特征均有一定的关联度。

准确率

准确率是最好理解的评价指标，它是一个比值：算法分类正确的数据个数/输入算法的数据的个数

但是使用准确率评价算法有一个问题，就是在数据的类别不均衡，特别是有极偏的数据存在的情况下，准确率这个评价指标是不能客观评价算法的优劣的。

我们检测对 1000 个人是否患有高血压进行检测，但是高血压的发病率不高，我们就假定这 1000 个人里面只有 100 个人真正患病。

此时，我们可以“无脑地”设计一个算法，就预测这 1000 个人全部健康。根据准确率的定义，除了那 100 个真正患病的病人我们预测错了，其余健康的 997 人我们的算法都预测正确了，此时

准确率 = \cfrac{算法分类正确的数据}{输入算法的数据的总数} = \cfrac{990}{1000} = 90%

事实上，这样的算法没有任何的预测能力，真的用于医疗是极其不负责的，于是我们就应该考虑是不是评价指标出了问题。这个时候就需要别的评价指标了。

精准率、召回率

精准率就是“预测为正例的那些数据里预测正确的数据个数”。

召回率就是“真实为正例的那些数据里预测正确的数据个数”。
​
F1-Score表示模型分数关于Precision score和Recall Score的函数。F1-score是一种机器学习模型性能指标，它赋予Precision score和Recall Score相同的权重，以衡量其准确性方面的性能，使其成为准确性指标的替代方案（它不需要我们知道样本总数）。

F1-Score = 2* 精确分数 * 召回分数/ （精确分数 + 召回分数/）

​
总结就是

  • Precision score用于衡量模型在预测为正实际为正占所有预测为正样本计数比例的能力。
  • Recall Score用于衡量模型性能，表示预测为正实际为正占所有实际为正样本计数比例的能力。
  • Accuracy Score用于衡量模型性能，以测量所有预测中真正和真负之和占所有样本计数的比率。
  • F1-Score是精度和召回率分数的谐波平均值，在选择精度或召回率分数可能导致模型分别给出高误报和漏报的情况下用作指标。

在本文中，由于高血压患少于普通人，因此采用准确率和f1分数衡量模型的效果

特征优选

使用RFE进行特征选择：RFE是常见的特征选择方法，也叫递归特征消除。它的工作原理是递归删除特征，
并在剩余的特征上构建模型。它使用模型准确率来判断哪些特征（或特征组合）对预测结果贡献较大。

结果显示特征与均值，方根幅值，波形因子和脉冲指数的关系最大，因此只采用这四个作为特征向量，构建模型
